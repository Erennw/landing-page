# Brewed — Modern Coffee Shop Landing Page (HTML, CSS, Vanilla JS)

A clean, modern, and responsive one-page coffee shop website. It features a hero section, signature menu, brand story, locations, and a newsletter form. Includes mobile navigation, dark mode, smooth scrolling, and accessible focus styles.

## Live Demo
https://erennw.github.io/landing-page/

## Features
- Semantic HTML5 layout and BEM class naming
- Responsive grid for menu and sections
- Sticky header with mobile navigation
- Dark mode with preference stored in localStorage
- Sample menu rendered via JavaScript
- Newsletter form with basic email validation
- Accessible focus outlines and keyboard-friendly controls
